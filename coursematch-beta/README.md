# Course Match Master - Beta Deployment

## Quick Start Guide

This folder contains everything you need to deploy your Course Match tutorial system.

## Files Included:
- `index.html` - Main landing page
- `readiness-check.html` - Technical readiness check page
- `tutorial.html` - Interactive tutorial with chatbot

## Deployment Options:

### Option 1: Netlify Drop (Easiest - 2 minutes)
1. Go to https://app.netlify.com/drop
2. Drag this entire folder onto the page
3. Get your instant link!

### Option 2: GitHub Pages
1. Create account at github.com
2. Create new repository
3. Upload all files
4. Enable Pages in Settings > Pages
5. Select "Deploy from a branch" and choose "main"

### Option 3: Vercel
1. Go to vercel.com
2. Sign up with GitHub
3. Import project and upload files
4. Deploy!

## Customization:
- Update the feedback email in index.html
- Modify colors by searching for #667eea in the files
- Add your name/credits in the footer
